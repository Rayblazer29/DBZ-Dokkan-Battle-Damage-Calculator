I didn't like how today's damage calculators looked, and some are even inaccurate, so I made my own.

In the hit mobile game Dragon Ball Z: DOKKAN BATTLE, damage receieved and delt can be a little confusing.

This web app I built aims to help users grasp a better understanding of how damage in this game works, 
as well as not have to lose a good run and several hours in-game because they took too much damage.

This tool allows users to enter in-game stats to calculate the estimated damage taken by the enemy.
It also includes a few preset stats of notorious hard-hitters.
